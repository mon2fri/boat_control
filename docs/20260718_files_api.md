# CSV File Upload and Header Inspection API

## Endpoints

### `POST /api/files/upload/`

Uploads two CSV files for comparison and returns header inspection results.

**Request:**
- Content-Type: `multipart/form-data`
- Fields:
  - `file_a`: First CSV file (required)
  - `file_b`: Second CSV file (required)

**Response (200):**
```json
{
  "file_a_path": "/path/to/uploads/abc123_file_a.csv",
  "file_b_path": "/path/to/uploads/def456_file_b.csv",
  "inspection": {
    "file_a_name": "file_a.csv",
    "file_b_name": "file_b.csv",
    "columns_a": ["id", "name", "value"],
    "columns_b": ["id", "name", "score"],
    "common_columns": ["id", "name"],
    "only_in_a": ["value"],
    "only_in_b": ["score"]
  }
}
```

**Errors:**
- `400`: Non-CSV file or file exceeds 500 MB limit
- `405`: Non-POST method

### `POST /api/files/inspect/`

Re-inspects previously uploaded files by path.

**Request:**
```json
{
  "file_a_path": "/path/to/uploads/abc123_file_a.csv",
  "file_b_path": "/path/to/uploads/def456_file_b.csv",
  "file_a_name": "original_a.csv",
  "file_b_name": "original_b.csv"
}
```

**Response (200):** Same inspection format as upload endpoint.

**Errors:**
- `400`: Missing required paths
- `404`: File not found

## Limits

| Limit | Value |
|-------|-------|
| Maximum file size | 500 MB |
| Maximum columns | 500 |
| Accepted file extension | `.csv` |
| Filename sanitization | UUID prefix + sanitized name |

## Filename Sanitization

Uploaded filenames are sanitized to prevent path traversal:
- UUID prefix added for uniqueness
- Non-alphanumeric characters replaced with `_`
- Hidden files (starting with `.`) get `file_` prefix
- Maximum length: 200 characters

## Encoding

Files are read using Polars with Arrow backend. Polars auto-detects encoding.
Binary files are treated as single-column CSV with raw bytes as header.

## Error Handling

- Invalid CSV format: Polars raises parse error wrapped in `ValueError`
- Missing files: `404` response with error message
- Path traversal attempts: Blocked by UUID prefix and sanitization
